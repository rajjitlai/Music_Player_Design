# 🎵 Premium Music Player Design

A sleek, responsive, and minimalist music player web interface built with modern web standards.

![Preview](img/music_back.jpg)

## ✨ Features
- **Responsive Design**: Optimized for desktops and mobile devices.
- **Interactive Controls**: Play/pause functionality with dynamic icon changes.
- **Ambient Background**: Glassmorphism-inspired overlay for a premium feel.
- **Custom Audio Support**: Easily adaptable to your favorite tracks.

## 🛠️ Technology Stack
- **HTML5**: Semantic structure and audio integration.
- **CSS3**: Advanced styling, gradients, and responsive layouts.
- **JavaScript**: Interactive playback controls and dynamic UI updates.

## 🚀 Quick Start
### 1. Clone the repository
```bash
git clone https://github.com/rajjitlai/Music_Player_Design.git
```
### 2. Open locally
Simply open `index.html` in your favorite browser.

## 📁 Project Structure
- `index.html`: The main entry point.
- `css/style.css`: Project styles and layout.
- `js/script.js`: Audio control logic.
- `img/`: High-quality assets and icons.
- `media/`: Background music files.
